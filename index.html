Parking Guide

御池地下駐車場

入庫
入口御池通を東向きに走行 → 烏丸交差点から2つ目の信号を左（西入口）
駐車位置B1 中央エリア（ゼスト側・東寄り）が地上への徒歩移動に便利
サイズ制限車高 2.1m ／ 車幅 2.0m ／ 車長 5.0m 以内
場内について
B1 ↔ B2スロープが2か所。B1に停めると階層移動が不要で楽
事前精算機各階に設置。精算後は15分以内に出庫レーンへ
📄 場内MAP（PDF） ／ 🌐 駐車場公式ページ
地上への出方（徒歩）
鳩居堂へ地下街（ゼスト御池）東端の出口・階段から地上へ → 寺町通すぐ
近又へ鳩居堂から寺町通を南下 → 御幸町通へ 徒歩約10分（駐車場には戻らず直行）
迷ったら地下街内の「御幸町通」「寺町通」の案内板を目印に。
出庫（山崎方面）
精算近又から戻り、場内の事前精算機で支払い → 15分以内に出庫
出口「西出口（堀川方面）」の看板に従い、スロープ後直進（右折NG）
ルート堀川御池を左折 → 堀川通南下 → 府道10号 → 大山崎（171号回避）
料金昼間上限 1,800円 クレカ・交通系IC・QR決済可

Destinations

各スポット

01
⛩ 北野天満宮
学問の神・菅原道真を祀る全国天満宮の総本社。梅の名所として名高く、2月の梅苑公開には多くの参拝者が訪れる。毎月25日には天神市が立つ
🕐 境内自由
上京区馬喰町 公式P 約300台
→ 公式サイト
🌸 平野神社への近道
北門（御前通沿い）を出て西へ徒歩約3〜4分 → 平野神社の東鳥居に到着
02
🌸 平野神社
60種・400本超の桜で知られる京都屈指の花見名所。珍種が多く、北野天満宮と隣接するため合わせて散策できる
🕐 6:00〜17:00
上京区西大路通一条上ル
北野天満宮の北門から徒歩3〜4分
→ 公式サイト
03
🫘 中村製餡所
大正元年創業100年超の餡専門店。職人手作りの粒あん・白あんを量り売りで購入でき、手土産に人気
🕐 8:00〜17:00（水・日定休）
上京区一条通御前西入大東町88
カード・Suica・PayPay可
📞 075-461-4481
→ 公式サイト
04
📜 鳩居堂 本店
1663年創業350年超、香と和文具の老舗。お香・和紙・便箋・ぽち袋など上質な品揃えが魅力の京都本店
🕐 10:00〜18:00
中京区寺町通御池下ル
→ 公式サイト
05
🍱 懐石 近又
1801年（享和元年）創業の老舗懐石料理店。国登録有形文化財の町家建築で、季節の食材を活かした京懐石をいただける
🕐 12:00〜（完全予約制） ✅ 12:00 予約済
中京区御幸町四条上ル
📞 075-221-8813
→ 公式サイト
06
🖼 大山崎山荘美術館
大正期の英国風山荘を転用した美術館。モネの「睡蓮」連作と、特別展「共鳴　河井寛次郎×濱田庄司」を同時に堪能できる。安藤忠雄設計の「地中の宝石箱」地中館も必見
🕐 10:00〜17:00（入館16:30まで）
乙訓郡大山崎町銭原5-3

→ 特別展「共鳴」詳細
→ 美術館公式サイト
